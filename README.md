This repository contains the static privacy policy website for the
**Expenses Log** Android application.

The site is implemented as a lightweight static HTML page and provides
localized versions of the privacy policy for all languages supported
by the application.

## Purpose

- Fulfill Google Play privacy policy requirements
- Provide a transparent, user-accessible privacy statement
- Ensure consistency across all supported locales

## Supported Languages

The privacy policy is available in the same set of languages as
the Expenses Log application (80+ languages).
